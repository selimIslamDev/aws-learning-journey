# Day 01 - AWS Fundamentals

## Date: 05 April 2026

## Topics Covered

### 1. Why AWS?
**বাংলা:** AWS হলো world এর সবচেয়ে বড় cloud provider। মানে সবচেয়ে বেশি company AWS use করে। তাই job market এ AWS জানলে কাজ পাওয়া সহজ। Netflix, Facebook, NASA — সবাই AWS use করে।

**English:** AWS is the #1 cloud provider in the world with 31% market share. Most companies use AWS, so learning it gives you the best job opportunities.

### 2. What is Cloud Computing?
**বাংলা:** ধরো তোমার computer এ একটা বড় software চালাতে হবে কিন্তু তোমার computer দুর্বল। Cloud মানে হলো internet এর মাধ্যমে অন্য কারো powerful computer use করা — নিজে কিনতে হবে না, শুধু ভাড়া দিয়ে use করবে।

**English:** Instead of buying your own servers, you rent computing power, storage, and services over the internet — pay only for what you use.

### 3. Public Cloud vs Private Cloud?
**বাংলা:** Public cloud মানে AWS, Google Cloud এর মতো — সবাই share করে use করে, সস্তা। Private cloud মানে শুধু একটা company এর জন্য আলাদা — অনেক secure কিন্তু দামি। যেমন bank বা government সাধারণত private cloud use করে।

**English:** Public cloud is shared infrastructure (like AWS) — cheap and scalable. Private cloud is dedicated to one organization — more secure but expensive.

### 4. Why is Public Cloud so Popular?
**বাংলা:** কারণ তিনটা — সস্তা, দ্রুত, এবং যেকোনো জায়গা থেকে access করা যায়। নিজে server কিনলে লক্ষ টাকা লাগে, কিন্তু AWS এ মাসে হাজার টাকায় কাজ হয়।

**English:** It's popular because of 3 reasons — Cost (no upfront investment), Speed (launch servers in minutes), and Scalability (grow or shrink anytime).

### 5. What is Virtualization?
**বাংলা:** ধরো একটা powerful computer আছে। Virtualization মানে সেই একটা computer কে ভাগ করে ১০টা আলাদা computer এর মতো বানানো। AWS এই কাজটাই করে — একটা বড় server কে অনেক ছোট ছোট virtual server এ ভাগ করে আমাদের দেয়।

**English:** Virtualization is the technology that splits one physical server into multiple virtual machines (VMs). This is the foundation of cloud computing — AWS uses this to give you your own "virtual computer" without owning physical hardware.
